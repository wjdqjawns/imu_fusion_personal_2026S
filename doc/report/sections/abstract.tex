\begin{abstract}
    This report presents the design and implementation of an Attitude and Heading Reference System using the MPU-9250 Inertial Measurement Unit. The study covers the full pipeline from sensor characterization to state estimation, including stochastic noise modeling via Allan Variance and Power Spectral Density analysis, static calibration of the accelerometer, and Hard/Soft Iron correction for the magnetometer.\\

    Three attitude estimation algorithms are designed and compared: a Complementary Filter, the Mahony and Madgwick filters, and an Extended Kalman Filter. The EKF is formulated with a quaternion-based state representation to avoid gimbal lock and provide globally stable attitude estimates. Algorithm performance is first validated through Python-based simulation under controlled motion profiles, then evaluated on hardware using an Arduino-based platform with the MPU-9250.\\
